This repository demonstrates my **full-stack website deployment on AWS**, hosting both a WordPress site and phpMyAdmin on an EC2 instance running Ubuntu Linux. The deployment was manually configured, giving hands-on experience in Linux server setup, cloud hosting, and database management.

## Project Overview

I successfully hosted a fully functional **WordPress website** and **phpMyAdmin** on an **AWS EC2 instance** using the **LEMP stack**:

- **Linux (Ubuntu)** for the operating system
- **Nginx** as the web server
- **MariaDB** for database management
- **PHP** for backend processing
- Manual configuration and optimization for high performance

## Key Features

- Fully functional **WordPress website** hosted on EC2
- **phpMyAdmin** configured for database management
- Secure and optimized **Nginx + PHP + MariaDB** setup
- Hands-on Linux server configuration and troubleshooting

## Tech Stack

- **OS:** Ubuntu Linux

- **Database:** MariaDB  
- **Backend:** PHP  
- **Applications:** WordPress, phpMyAdmin  
- **Cloud Platform:** AWS EC2

## How to Run / Replicate

1. Launch an **EC2 instance** (Ubuntu) in your AWS account.
2. Install **Nginx, PHP, MariaDB**.
3. Configure your security groups to allow HTTP/HTTPS and SSH.
4. Set up WordPress and phpMyAdmin manually.
5. Test and verify that your website and database management tool are running properly.
